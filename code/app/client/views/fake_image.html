Fake Image

-Social media platforms such as twitter have a limit on the number of words that can be used for posting. In such situations, people often resort to using images for spreading information.
-In such cases, images become another potential source for spreading fake news as images can be tampered with, in order to spread misinformation.
- Thus this module detects whether a given image is potentially fake. To identify tampered images, the metric identifies if the image has been tampered with or not.
- The metric also highlights the regions that have been altered
